## ℹ️ About project

This repository represents the compilation of knowledge and projects developed throughout the Ford Enter Frontend course.
The goal of this final project is to consolidate and present in an integrated way the work completed during the sprints, showcasing the progress and practical application of the technologies and methodologies learned.

The final project brings together the different stages of web application development:

Fundamentals of version control with Git and GitHub

Building static and interactive web pages with HTML, CSS, and JavaScript

Developing a complex application using Angular, TypeScript, and Bootstrap

Integration of an internal dashboard system

Each previous sprint focused on specific aspects of frontend development, progressively building the necessary skills for this final project. The Final Sprint is not an isolated new project, but rather the integration of each stage.

## 🔗 How the sprints are connected

The development track was divided into several sprints, each with specific objectives:

*   **Sprint 2 (Desafio_da_Sprint2):**
  *   Introduction to Git, GitHub, and the SCRUM methodology
  *   Establishing the foundation for version control
  *   Hands-on practice with version control workflows

*   **Sprints 4, 5 e 6 (Desafio_da_Sprint4, Desafio_da_Sprint5, Desafio_da_Sprint6):** These sprints focused on the progressive development of a website for Ford, specifically centered around the Ford Ranger, and later expanded into a concept for the official Ford Brazil website. Responsiveness was carefully considered from the beginning.


*   **Sprint 7:**
*   Application development using Angular, TypeScript, and Bootstrap for the frontend

*   Backend with Node.js and Express.js

*  Internal dashboard for Ford employees

*   User authentication, vehicle metrics display, and VIN lookup

:arrow_right: The `Sprint-Final` repository serves as a point of aggregation and documentation of this evolutionary journey, where each sprint contributed with a set of skills and a software artifact that, together, illustrate a comprehensive learning experience in frontend development.

#### In this current sprint, the integration of the two applications was carried out, leading to the complete migration of the website system to Angular.

## 📝 Sobre o Projeto

Este repositório representa a compilação dos conhecimentos e projetos desenvolvidos ao longo do curso de Frontend da Ford Enter. O objetivo deste projeto final é consolidar e apresentar de forma integrada o trabalho realizado nas sprints, demonstrando a evolução e a aplicação prática das tecnologias e metodologias aprendidas.

O projeto final agrega as diferentes fases de desenvolvimento de aplicações web:
- Fundamentos de controle de versão com Git e GitHub
- Construção de páginas web estáticas e interativas com HTML, CSS e JavaScript
- Desenvolvimento de uma aplicação complexa utilizando Angular, TypeScript e Bootstrap
- Integração de um sistema de dashboard interno

Cada sprint anterior focou em aspectos específicos do desenvolvimento frontend, construindo progressivamente as competências necessárias para a realização deste projeto. A Sprint Final não é um projeto novo isolado, mas sim a integração de cada etapa.

## 🔄 Evolução das Sprints

A trilha de desenvolvimento foi dividida em várias sprints, cada uma com objetivos específicos:

### Sprint 2 - Fundamentos de Versionamento
- Introdução ao Git, GitHub e metodologia SCRUM
- Estabelecimento da base para controle de versão
- Prática de fluxo de trabalho de versionamento

### Sprints 4, 5 e 6 - Desenvolvimento Web Frontend
- **Sprint 4**: Desenvolvimento frontend com HTML5, CSS3 e JavaScript básico
  - Design mobile-first, interatividade e semântica
  - Website de apresentação da Nova Ford Ranger 2022
  - Seções de especificações e formulário de contato

- **Sprint 5**: Aprimoramento do website da Ford Ranger
  - Maior interatividade e validações JavaScript nos formulários
  - Melhoria na experiência do usuário e integridade de dados

- **Sprint 6**: Recriação de um conceito para o site oficial da Ford Brasil
  - Recursos mais ricos como carrossel de imagens interativo
  - Página de lançamentos com comparações de veículos
  - Formulário de contato com validações e considerações para LGPD

### Sprint 7 - Aplicação Full-Stack
- Desenvolvimento de aplicação usando Angular, TypeScript e Bootstrap no frontend
- Backend com Node.js e Express.js
- Dashboard interno para funcionários da Ford
- Autenticação de usuários, exibição de métricas de veículos e consulta de VIN

### Sprint Final - Integração
- Migração completa do sistema de website para Angular
- Integração das duas aplicações (website e dashboard)
- Consolidação de todas as funcionalidades em uma única aplicação Angular

## ✨ Funcionalidades

### Estrutura e Estilização de Website
- Layouts responsivos
- Headers, footers, seções de conteúdo, galerias de imagens e formulários
- Design adaptável para dispositivos móveis e desktop

### Interatividade com JavaScript
- Menus mobile
- Carrosséis de imagens com navegação automática e manual
- Efeitos hover e manipulação dinâmica de conteúdo

### Validação de Formulários
- Validações client-side usando JavaScript
- Verificações de campos obrigatórios, formatos de email, números de telefone (com máscara)
- Aceitação de termos (LGPD) com feedback imediato aos usuários

### Consumo de API e Gerenciamento de Dados
- Comunicação com API backend para autenticação de usuários
- Recuperação de dados de veículos
- Apresentação dinâmica de informações em um dashboard

### Autenticação de Usuários
- Sistema de login com email e senha
- Armazenamento de token de sessão (localStorage)
- Persistência de autenticação na aplicação Angular

### Dashboard Interativo
- Painel de controle com seleção de veículos
- Exibição de métricas (vendas, veículos conectados, atualizações pendentes)
- Busca por VIN e atualizações dinâmicas da interface baseadas nas interações do usuário

### Navegação e Roteamento
- Sistema de roteamento do Angular para criar uma Single Page Application
- Múltiplas views (Login, Home, Dashboard)
- Navegação suave entre as páginas

## 🚀 Como Executar


### Instalação

1. Clone o repositório
   ```bash
   git clone https://github.com/Amaral-Gabriel/Sprint-Final.git
   cd Sprint-Final
   ```

2. Instale as dependências
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento
   ```bash
   ng serve
   ```

4. Acesse a aplicação em seu navegador
   ```
   http://localhost:4200/
   ```

5. Para o backend (se necessário)
   ```bash
   # Em outro terminal
   json-server --watch db.json
   ```
